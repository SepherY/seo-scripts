Column Legend
URL — Crawled page URL. REDIR badge = Angular client-side redirect detected.
Title — document.title after JS rendering. NO-TITLE flag if missing.
H1 — First <h1> tag text. Badge shows count if multiple H1s. NO-H1 flag if missing.
Text — document.body.innerText.length. EMPTY = 0 chars (render failure). THIN = under 400 chars.
Robots — <meta name="robots"> content. Flags: index (green), noindex (red), nofollow (orange).
Canonical — <link rel="canonical">. "self" (green) = self-referential. Other URL = non-self (orange).
Footer — stb-footer-links element present in DOM. Structural component, NOT the SEO content signal (use API column for that).
API — Article API (/api/v2/article/item) detected via Performance API. "yes" = SEO article content loaded. "no" = no article content.
Iframes — Iframe domains detected by MutationObserver (Action hook) + DOM scan. Hover for full URLs.
Links — Internal / external <a href> count from rendered DOM.
Imgs — <img> element count. Does not include CSS background images.
Gate — DOM Stability Gate timing. The Action snippet holds SF's rendering pipeline via a Promise until Angular hydrates (innerText > 400 chars + 2 stable readings). Shows resolution time in ms. No gate = snippet didn't execute on this page.
Err — Console error count. console.error override captures errors before Angular boots. Hover for first error message.
Status — HTTP status code from PerformanceNavigationTiming.responseStatus. JS badge = client-side redirect (pushState or replaceState).
Res Fail — Resources that failed to load (responseEnd = 0). Hover for URLs and durations. Could be blocked by CSP, CORS, or network errors.
Res Slow — Resources taking >2s to load. Hover for URLs and durations. May indicate CDN issues or large payloads.
TTFB — Time to First Byte from navigation timing. High values (>500ms) suggest server-side delays.
Heap — JS heap memory usage in MB. High values may indicate memory leaks in Angular or third-party scripts.
Debug Columns (toggle with SHOW DEBUG)
Angular — Angular version from ng-version attribute.
Words — Word count from innerText.split(/\s+/). More accurate than char count for content assessment.
Scripts — Number of <script> elements. High counts may affect rendering performance.
Classes — document.body.className. Useful for identifying page variants (e.g. "mainDomain").
Stability Log — Full gate timeline: text length + footer status at each 500ms poll until resolution. Hover for full log.
Console Errors — Full pipe-delimited error messages. Hover for complete text.
Meta Desc — First 40 chars of meta description content. Hover for full text (up to 160 chars).
